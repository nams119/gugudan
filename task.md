# 구구단 시각화 학습 앱 개발 작업 목록

* [/] 1단계: 기본 UI 및 상태 관리 설정 (스트림릿 뼈대)
  * [ ] 앱 제목 및 모바일 레이아웃 설정
  * [ ] 세션 상태(`st.session_state`) 초기화 (현재 문제, 보기 3개, 정답 여부 등)
  * [ ] 학습 모드 선택 UI (1~9단 지정 또는 전체 랜덤)

* [ ] 2단계: 문제 생성 로직 구현
  * [ ] 설정된 모드에 따라 랜덤으로 구구단 문제(A x B) 생성
  * [ ] 정답과 오답 2개를 포함한 3개의 객관식 보기 생성
  * [ ] 보기 순서 섞기

* [/] 3단계: 시각화 애니메이션 컴포넌트 개발 (핵심 기능)
  * [ ] HTML/CSS/JavaScript를 활용한 N x M 그리드 애니메이션 렌더링 함수 작성
  * [ ] 파이썬에서 문제(A, B)와 표시할 이모지(🍎, 🐶 등 랜덤)를 HTML로 전달
  * [ ] 뾰로롱 나타나는 순차적 등장 애니메이션 효과 (CSS Transition/Animation)

* [ ] 4단계: 정답 확인 및 게임 흐름 제어
  * [ ] 사용자가 보기를 클릭했을 때 정답을 확인하는 로직 (Callback 함수)
  * [ ] 정답 시 칭찬 이펙트 (풍선 효과 `st.balloons()` 등) 표시
  * [ ] '다음 문제' 버튼 구현 및 상태 업데이트
